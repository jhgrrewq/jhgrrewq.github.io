es6 class 经 babel 编译分析

发表于 2018-03-14

参考: 揭秘babel的魔法之class魔法处理、揭秘babel的魔法之class继承的处理2

class

class 声明经过 babel 编译后其实是定义一个立即执行函数，返回定义好的构造函数。重点在于理解立即执行函数内部调用的 _classCallCheck() 和 _createClass() 方法

_classCallCheck()

这个函数确保 class 不能作为一般的函数调用，因为只有 new 构造函数，this 才会指向创建的对象实例

_createClass()

该方法定义一个立即执行函数，返回一个方法，可以对构造函数本身和其原型扩展属性和方法（内部实质调用 Object.defineProperty()）

class 继承

class 继承经过 babel 编译后其实子类构造函数是定义一个立即执行函数，传入父类构造函数，返回定义好的构造函数。重点在于理解立即执行函数内部调用的 _inherits() 和 __possibleConstructorReturn() 方法

_inherits()

该方法实现了对父类原型链属性的继承, 实质是调用 Object.create() 方法实现对父类的原型继承，同时调用 Object.setPrototypeOf() 方法实现对父类静态属性方法继承

_possibleConstructorReturn()

该方法传入的第二个参数本质是调用 父类.call(this, props)
